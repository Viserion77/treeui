Add `@treeui/react`, an early React component package built on the same `@treeui/tokens` and `t-*` BEM classes as `@treeui/vue`. The first release ships `TButton`, `TInput`, `TBadge`, and `TCard`, plus the shared `TSize`, `TVariant`, `TCardVariant`, and `TBadgeTone` types. Import `@treeui/react/style.css` once to load tokens, themes, and component styles.

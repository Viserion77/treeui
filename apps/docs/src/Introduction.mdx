<Title>TreeUI</Title>
<Subtitle>Clean components for modern Vue products.</Subtitle>

TreeUI is a Vue 3-first component library with a framework-ready foundation.
The project started with a deliberately small public core and now expands that base with app-oriented primitives,
while keeping strong accessibility defaults and a token system that stays portable beyond Vue.

As a design system, TreeUI organizes work into foundations, patterns, components, and products.
That structure helps teams move faster while keeping visual language and behavior consistent.

The architecture follows a tree model internally:
Roots for tokens and principles, Trunk for utilities and contracts, Branches for framework packages,
and Leaves for documentation and examples.
